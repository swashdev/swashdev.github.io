This is my website.

At one time, this web address was intended to be used to host a portfolio of my work. However, several years later, I have gone through a lot of changes in my life and the use-case for it doesn't seem to be there anymore.

I've thought about using this website as a blog, but I don't have much going on right now, and I would rather use the time that I would spend writing about myself making things instead. Perhaps I'll write some stories, or maybe when I get back into the gamedev rhythm I'll bring the portfolio back with a fresher feel. Who can say what the future holds?

Specific pages like the FYL homepage will still be here, and I don't intend to delete my portfolio per se, but I don't really know what I'm going to do with this website in the near future beyond using it as a sandbox for certain web projects. This page will stay for now, partly so I can park this domain and not have it redirected to a casino website again. Thanks for that, by the way, GitHub.

Life wouldn't seem tragic were it not also beautiful.
